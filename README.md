IDLC Finance PLC. - Q1 2026 Financial Data Analytics & BI Project

A comprehensive data analytics portfolio project based on IDLC's un-audited financial statements. This project highlights financial data cleaning, dynamic Excel modeling, and executive dashboard design.

 Core Insights Formulated
Total Operating Income:** BDT 2.44 Billion (Consolidated)
Total Portfolio Assets:** BDT 180.25 Billion (Consolidated)
Net Profit After Taxation (NPAT):** BDT 622.48 Million (Consolidated)
Risk Quality Tracker:** Asset NPL risk benchmarking against industry variables.

 Project Deliverables
